## ✨ Features

*   **🎨 Preset Browser:** A scrollable list of pre-installed, community-favorite WezTerm themes (Dracula, Tokyo Night, Catppuccin, etc.) to use as a starting point.
*   **🌊 Vibe Generator:** Pick **one** seed color, and the tool uses HSL color math to generate a full, harmonious 16-color palette, background, and foreground that matches that specific energy.
*   **🎲 Randomizer:** For when you want to let fate decide your aesthetic.
*   **👀 Dual-Pane Preview:**
    *   **Galaxy Grid:** A dense color grid (mimicking the WezTerm docs) to see how ANSI colors interact with the background.
    *   **Live Code Preview:** A syntax-highlighting pane that updates instantly when you click grid cells, letting you check readability for coding.
*   **🔤 Font Manager:** Preview the top 10 most popular monospace fonts (JetBrains, FiraCode, etc.) and generate the specific installation commands for MacOS, Windows, or Linux.
*   **🚀 Instant Export:** Click anywhere to copy the code. Supports three export formats:
    *   `Builder` (for `config.colors = { ... }`)
    *   `Inline` (for `return { ... }`)
    *   `File` (for external modules)

## 🚀 Getting Started

There is no build process, no `npm install`, and no server required.

1.  **Download** the `index.html` file from this repository.
2.  **Open** it in any modern web browser (Chrome, Firefox, Edge, etc.).
3.  **Create** your theme.
4.  **Click** the grid to copy the Lua code.
5.  **Paste** it into your `.wezterm.lua`.

## 🛠️ How to use the Output

The tool defaults to the **Builder** format, which is the standard for most user configs.

1.  Open your WezTerm config file (usually `~/.wezterm.lua` or `C:\Users\You\.wezterm.lua`).
2.  Paste the copied code block:

```lua
local wezterm = require 'wezterm'
local config = wezterm.config_builder()

-- PASTE THE COPIED CODE HERE --
config.font = wezterm.font 'JetBrains Mono'
config.colors = {
    -- ... colors generated by the tool ...
}

return config
```
